## 飞行大战（pygame）

一个使用 `pygame` 编写的纵版射击小游戏，支持多敌机波次、生命值系统、得分机制、爆炸动画和素材集中管理。项目遵循 PEP 20 “The Zen of Python” 所倡导的清晰、模块化设计，核心逻辑拆分在 `game/` 包中，便于维护与扩展。

### 功能概览
- **英雄飞机**：左右移动、发射子弹，拥有5点生命值，被击中扣血1点，生命值归零时播放爆炸动画并显示游戏结束。
- **敌机体系**：支持3种类型敌机同屏生成，每种敌机拥有不同生命值、移动速度和分值，被击中扣血，生命值归零时播放爆炸动画并重置。
- **得分机制**：击毁敌机获得相应分值，分值根据敌机类型提升。
- **资源管理**：`GameAssets` 集中加载图片素材，减少重复 IO。
- **输入系统**：同时支持离散按键事件和持续按键状态，确保操作顺畅。
- **碰撞检测**：英雄与敌机/子弹之间的碰撞采用 `pygame.Rect` 检测，触发扣血、得分或游戏结束。
- **UI 显示**：实时显示英雄生命值、当前得分和敌机生命值条。
- **游戏状态**：支持游戏结束后的重新开始（R键）或退出（Q键）。
- **帧率稳定**：使用 delta time 驱动移动，确保游戏在不同设备上以60 FPS稳定运行。

### 环境要求
- Python 3.10+ (Python 2.7 is not supported)
- `pygame` >= 2.5

### 安装与运行
```bash
python -m venv .venv
source .venv/bin/activate          # Windows 使用 .venv\Scripts\activate
pip install pygame

python app.py
```

### 项目结构
```
airplane-game-master/
├── app.py                 # 入口：初始化、主循环
├── game/
│   ├── __init__.py
│   ├── assets.py          # 素材加载
│   ├── constants.py       # 全局常量
│   ├── hero.py            # 英雄飞机
│   ├── enemy.py           # 敌机逻辑
│   ├── bullet.py          # 子弹实体
│   ├── collision.py       # 碰撞处理
│   └── input_handler.py   # 键盘监听
└── feiji/                 # 游戏素材
```

### 操作说明
- `← / A`：向左移动
- `→ / D`：向右移动
- `Space`：发射子弹
- `R`：游戏结束后重新开始
- `Q`：游戏结束后退出游戏
