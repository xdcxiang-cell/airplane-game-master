## 飞行大战（pygame）

一个使用 `pygame` 编写的纵版射击小游戏，支持英雄飞机移动/射击、多架敌机多波次作战、子弹碰撞判定、爆炸动画、音效和素材集中管理。项目遵循 PEP 20 “The Zen of Python” 所倡导的清晰、模块化设计，核心逻辑拆分在 `game/` 包中，便于维护与扩展。

### 功能概览
- **英雄飞机**：左右移动、发射子弹，初始5条命（用爱心符号显示），开始时有护盾，每击杀5架敌机恢复护盾。被击中后先失去护盾，护盾消失后扣生命值。
- **敌机与子弹**：多架敌机多波次作战，支持多种敌机类型（不同生命值、速度和积分），敌机水平往返移动并按概率发射子弹，子弹越界自动清理，命中英雄触发护盾消失或生命值减少。
- **资源管理**：`GameAssets` 集中加载图片素材和音效，减少重复 IO。
- **输入系统**：同时支持离散按键事件和持续按键状态，确保操作顺畅。
- **基础碰撞**：英雄与敌机/子弹之间的碰撞采用 `pygame.Rect` 检测。
- **音效支持**：发射子弹和敌机爆炸时有相应音效。
- **积分系统**：击杀敌机获得积分，积分实时显示。

### 环境要求
- Python 3.9+
- `pygame` >= 2.0.2

### 安装与运行
```bash
python -m venv .venv
source .venv/bin/activate          # Windows 使用 .venv\Scripts\activate
pip install pygame

python app.py
```

### 项目结构
```
airplane-game-master/
├── app.py                 # 入口：初始化、主循环
├── game/
│   ├── __init__.py
│   ├── assets.py          # 素材加载
│   ├── constants.py       # 全局常量
│   ├── hero.py            # 英雄飞机
│   ├── enemy.py           # 敌机逻辑
│   ├── bullet.py          # 子弹实体
│   ├── collision.py       # 碰撞处理
│   └── input_handler.py   # 键盘监听
└── feiji/                 # 游戏素材
```

### 操作说明
- `← / A`：向左移动
- `→ / D`：向右移动
- `Space`：发射子弹

### 游戏规则
- 玩家初始拥有5条命，用爱心符号显示在屏幕左上角。
- 玩家开始时有护盾，护盾未被打破时受到攻击不会扣生命值。
- 每击杀5架敌机可以恢复一次护盾。
- 不同类型的敌机有不同的生命值和积分，击杀后获得相应积分。
- 积分实时显示在屏幕右上角。
- 敌机有血条显示，方便玩家了解其剩余生命值。
