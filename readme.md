# 飞机大战游戏

这是一个基于Pygame开发的经典飞机大战游戏，玩家控制英雄飞机射击敌人，避免被敌人攻击。

## 游戏玩法

### 控制方式
- **键盘方向键**：控制英雄飞机的移动（上、下、左、右）
- **空格键**：发射子弹
- **ESC键**：退出游戏

### 游戏目标
- 消灭尽可能多的敌人飞机
- 避免被敌人或敌人子弹击中
- 获得高分

### 游戏规则
- 英雄飞机初始有3条生命
- 敌人飞机随机从屏幕上方出现
- 敌人会随机发射子弹
- 击中敌人获得分数
- 被敌人或敌人子弹击中失去一条生命
- 所有生命耗尽游戏结束

## 游戏功能

### 1. 英雄飞机
- 可以上下左右移动
- 可以发射子弹
- 被击中后会爆炸并失去一条生命
- 生命耗尽后游戏结束

### 2. 敌人飞机
- 随机从屏幕上方出现
- 自动向下移动
- 随机发射子弹
- 被击中后会爆炸并消失

### 3. 子弹系统
- 英雄和敌人都可以发射子弹
- 子弹自动移动（英雄子弹向上，敌人子弹向下）
- 子弹击中目标后消失
- 子弹超出屏幕后消失

### 4. 碰撞检测
- 英雄与敌人碰撞
- 英雄与敌人子弹碰撞
- 英雄子弹与敌人碰撞

### 5. 音效系统
- 射击音效
- 爆炸音效
- 游戏结束音效

### 6. 分数系统
- 击中敌人获得分数
- 分数显示在屏幕顶部

### 7. 生命系统
- 初始3条生命
- 生命显示在屏幕顶部
- 生命耗尽游戏结束

## 如何运行游戏

### 环境要求
- Python 3.x
- Pygame库
- NumPy库（用于音效生成）

### 安装依赖
```bash
pip install pygame numpy
```

### 运行游戏
```bash
python app.py
```

### 调试模式
要启用调试模式，可以使用以下命令：
```bash
python app.py --debug
```
调试模式会显示更多游戏信息，如坐标、速度等。

## 开发说明

### 项目结构
```
airplane-game-master/
├── feiji/          # 游戏资源文件（图片、音效）
├── game/           # 游戏核心代码
│   ├── __init__.py
│   ├── assets.py   # 资源加载
│   ├── bullet.py   # 子弹类
│   ├── collision.py # 碰撞检测
│   ├── constants.py # 游戏常量
│   ├── enemy.py    # 敌人类
│   ├── hero.py     # 英雄类
│   └── input_handler.py # 输入处理
├── test/           # 测试用例
│   ├── test_bullet.py
│   ├── test_collision.py
│   ├── test_enemy.py
│   └── test_hero.py
├── app.py          # 游戏入口
└── README.md       # 本文件
```

### 代码结构
- **app.py**：游戏主入口，负责初始化游戏、处理事件循环
- **game/hero.py**：英雄飞机的实现
- **game/enemy.py**：敌人飞机的实现
- **game/bullet.py**：子弹的实现
- **game/collision.py**：碰撞检测的实现
- **game/input_handler.py**：输入处理的实现
- **game/assets.py**：资源加载的实现
- **game/constants.py**：游戏常量的定义

### 测试用例
测试用例位于test/目录下，可以使用pytest运行：
```bash
pytest test/
```
